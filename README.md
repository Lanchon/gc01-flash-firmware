# gc01-flash-firmware

FNRISI GC-01 Firmware Flasher for Linux

```
usage: gc01-flash-firmware [firmware-file] [gc01-block-device]
```

```
$ sudo ./gc01-flash-firmware fw.bin /dev/sd[x]
verifying firmware file size...
verifying block device size...
copying device to temporary image...
creating loop device...
mounting loop device...
verifying filesystem label...
verifying filesystem contents...
writing firmware...
unmounting loop device...
destroying loop device...
writing image back to device...
```
